In order for this project to launch correctly, you must set up the service
configuration information. Please use the sample files as an example.
1) Copy ServiceConfigurationSample.Cloud.cscfg to ServiceConfiguration.Cloud.cscfg
2) Fill in the file with the configuration parameters you want to use in the cloud
3) Repeat steps 1 and 2 for ServiceConfigurationSample.Local.cscfg for what settings
   you want when running the service locally
